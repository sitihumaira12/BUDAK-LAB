Drop photos here for the page 2 "Reel" montage. Each slide looks for
one of these exact filenames — if a file isn't there yet, that slide
just shows a themed icon tile instead, so the page never breaks:

  dashboard.jpg      -> Dashboards
  gis.jpg             -> GIS
  ai.jpg               -> Artificial Intelligence
  web.jpg              -> Website Development
  data.jpg             -> Data Analytics
  research.jpg         -> Research
  achievements.jpg     -> Achievements

Landscape photos, roughly 1600x900 or wider, work best (the frame is
16:9 on desktop). To rename the slides or point to different files,
edit the REEL_ITEMS list at the top of members.js.
